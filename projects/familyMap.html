Family Map


About

An Android application used to simulate family history. Upon registering, family data was created and google maps displayed where life events took place for all ancestors. Both front end and back were made from scratch. Thorough regression testing was done using JUnit.

Languages Used

  • Java
  • Android
  • Bash
  • SQLite

Technologies Used

  • Google maps API
  • Gson
  • JUnit tests
  • SQLite Database
  • Git

Design Patterns

  • Facade Pattern
  • Proxy Pattern
  • DAO Pattern
