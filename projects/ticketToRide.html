Ticket to Ride


About

In a group of four, my team and I created an Android app version of the multi-player board game Ticket to Ride. I was responsible for implementing the server and other communication portions of the project. We carefully read the specifications before implementing a phase and received high marks on all the phases of the project.

All clients in a game observed the game's events in real time. We implemented a game history feature and chat feature. The server was able to reboot after losing power mid game and the players were able to continue their games seamlessly.
