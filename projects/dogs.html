The Doggo Quiz


About

This project is a quiz which I made using the Vue plugin to dynamically bind the user interface to the underlying javascript file. It asks three highly insightful questions and based on the user's answers it will suggest the best dog breed for that user.

This uses an open source dog api for fetching images of different breeds.
