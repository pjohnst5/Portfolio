About

In this project, I developed a program in C to calculate the mandelbrot set in parallel using OpenMP. To do this I had to declare private OpenMP variables and keep track of position in the image in order to write the appropriate data to a file which was being accessed by multiple threads.

With one thread, the total computing time was ~24 seconds. With eight threads the computing time dropped to about 9 seconds. This shows the importance of parallelizing when appropriate and how it can cut computing time down drastically.
