About

Full paper

Raven's Progressive Matrices is a series of cognitive tests where a subject is given an incomplete matrix of images and is asked to determine which available answer image best completes the matrix pattern. The agent here uses a visual AI approach that combines both the Affine and Set Transformation Induction (ASTI) model (Kunda et. al., 2013) as well as Dark Pixel Ratio and Intersection Pixel Ratio (DPR/IPR) model (Joyner et. al., 2015) with some modifications to both models.

In a nutshell, the agent uses OpenCV to perform various affine transformations on the given images, performing a similarity function at each transformation. With each transformation the agent performs a sliding window of 10x10 pixels to account for any image jitters, making the best match possible. It then explores all the matches and chooses the best one, then performing that transformation on the candidate image to produce a prediction image. This is the ASTI portion of the agent.

If the certainty from the ASTI portion of the agent is not within a certain threshold, the agent then proceeds to compare the ratios of the dark-pixels between the given images. Each comparison casts a vote prportional to the dark-pixel ratios and the image with the highest vote is then selected as a candidate image. This candidate image is then compared to all the possible answer images and the image with the highest ratio vote is selected.
