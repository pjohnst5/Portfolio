Datalog Parser


About

A datalog parser useful for storing information and joining existing tables to create new ones. I used graph theory to convert the rules list into a graph and then foung the strongly connected components of the rules. This allowed for optimization when running the rules because I would only run the rules that were strongly connected to on another. Able to process more than 400,000 rows of information in less than a minute.

I wrote a bash script to run test cases using multiple test files

Languages Used

  • Bash

Algorithms Used

  • Topological Ordering
